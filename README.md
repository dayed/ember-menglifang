# Ember Menglifang

基于Ember开发的基础控件库。支持的浏览器：Google Chrome、Firefox、Safari、IE8+。

## 指南

### 使用grunt进行编译 (see http://gruntjs.com/getting-started)
```
$ npm install -g grunt-cli
$ npm install -g bower
$ npm install
$ bower install
$ grunt build
```

### 查看示例

```bash
$ cd examples/
$ node db-setup
$ node app

# 通过浏览器访问：http://localhost:8000
```
